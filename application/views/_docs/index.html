A) HTML Structure - top

The markup for this page is fairly standard. I have commented out each section to help you out.

There is also a newsletter input and submit field at the bottom. These are not hooked up to any specific service, so be sure that you either remove it or set it up with your email list service.

B) CSS Files and Structure - top

There is one main 'screen' css file named "horizon.css". This contains everything you need.

The CSS is broken up into sections which are defined in the comments.

There is a CSS file named animate.css that contains all the animations that you'll ever need.

The media queries are located at the bottom of the stylesheet. Check out the screenshot below so you can easily recognize them.
If you need to change anything, that's where you will find it. It's just standard CSS, like the rest of the CSS file.

C) JavaScript - top

This theme imports two JavaScript files.

  1. jQuery Latest From google
  2. bootstrap-min.js - Main Scripts
